# Motion Data

**File Extensions**: .msa

## Description
The MotionData script format is a subset of Script Format, it contains
information about 3D animation properties, combat and effect properties


## Composition
**Example File**: [motion.msa](files/motion.msa)


## Utility
You can handle this format with [simple-libjigen](https://github.com/OriDevTeam/simple-libjigen)
if you need to convert to other formats, unify or partialize data
